Summary: Latitude vs. Weather

500 cities were randomly selected from around the world using latitudes and longitudes. City names were then collected using citipy. Cities names were applied in Weather API to obtain weather details such as humidity, temperature, Cloudiness and wind speed. This information was gathered to analyze the effects of latitudes on the weather patterns. A series of scatter plots and linear regression were used to plot Temparature, Humidity %, Cloudiness% and Wind Speed Vs Latitudes. A few observations were then drawn about the trends of the weather using the equator as a location reference.
